# 📘 Estudos em Python

Repositório dedicado ao armazenamento de arquivos, exercícios e projetos desenvolvidos ao longo dos meus estudos de Python.

## 📚 Base de estudo

Este repositório segue como base o livro **"Curso Intensivo de Python"**, de Eric Matthes.

## 🎯 Objetivo

Organizar e acompanhar minha evolução no aprendizado da linguagem Python, praticando conceitos fundamentais e desenvolvendo pequenos projetos.

## 📂 Estrutura

Os arquivos estão organizados conforme o progresso dos estudos, podendo incluir:

* Exercícios práticos
* Anotações
* Pequenos projetos
* Testes de código

## 🚀 Tecnologias utilizadas

* Python

Este repositório está em constante atualização conforme avanço nos estudos.
